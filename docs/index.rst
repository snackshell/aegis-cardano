Overview
--------

Aegis is a comprehensive security solution for the Cardano blockchain that provides:

- **Real-time Security Analysis**: Monitor addresses, transactions, and assets for potential security threats
- **AI-Powered Risk Assessment**: Leverage machine learning to identify suspicious patterns and behaviors
- **Multi-Platform Support**: Access security services through REST API, Telegram bot, and Discord bot
- **Transaction Explanation**: Understand complex transactions in plain language
- **Asset Verification**: Verify the legitimacy of Cardano assets and policies
